Subscription Tracker 🔔
A subscription management app with analytics and renewal reminders. Built with a modern tech stack for secure, efficient tracking of recurring payments.

Tech Stack ⚙️
Backend
Node.js | Express | MongoDB | JWT | Nodemailer
Frontend
React | TypeScript | Tailwind CSS | Zustand | Tanstack React Query

Features ✨
🔐 Secure JWT authentication

📊 Subscription analytics dashboard

✉️ Email reminders (configurable days before renewal)

🔄 Auto-renewal date calculation

📱 Mobile-responsive UI

📈 Spending trends visualization
